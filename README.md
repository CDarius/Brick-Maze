# Brick Maze Control Board

Firmware project for the Brick Maze control board, built with PlatformIO for the Seeed Studio XIAO ESP32S3.

## Peripherals

- 64x8 NeoPixel display
- Two Kitten 2Kg servo
- MAX98357 audio out
- Serial connection with the AHRS controller

## MCU

- Seeed Studio XIAO ESP32S3
